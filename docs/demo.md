# Demo Script

1. Approve a customer
   - Go to Customers and set KYB to approved.

2. Whitelist an address
   - Add the wallet address to the customer's whitelist.

3. Create a mint request
   - Use Mint & Redeem to create a mint request.

4. Approve mint (maker/checker)
   - Approve the request using a second user.

5. Submit on-chain
   - Submit to chain and show the transaction hash in the UI.

6. Freeze address
   - Use Compliance to freeze the address and note it is now blocked.

7. Redeem tokens
   - Create and submit a redeem request.

8. Export audit log CSV
   - Download audit logs from the Audit Log page.
